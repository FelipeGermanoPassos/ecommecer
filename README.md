# E-commerce Simples

Um projeto de e-commerce desenvolvido com HTML, CSS e JavaScript puro, sem frameworks.

## Funcionalidades

✅ **Catálogo de Produtos**

- Exibição de produtos em grid responsivo
- Imagens, descrições e preços
- 12 produtos mockados de exemplo

✅ **Carrinho de Compras**

- Adicionar/remover produtos
- Ajustar quantidades
- Cálculo automático do total
- Persistência com localStorage

✅ **Sistema de Favoritos**

- Marcar produtos como favoritos
- Visualizar apenas favoritos
- Contador de favoritos no header
- Persistência com localStorage

✅ **Filtros**

- Filtro por categorias (Eletrônicos, Áudio, Wearables, etc.)
- Filtro por faixa de preço (mínimo e máximo)
- Botão para limpar todos os filtros

✅ **Checkout Fake**

- Formulário de dados do cliente
- Seleção de forma de pagamento
- Confirmação de pedido com número gerado
- Modal de sucesso

## Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Navegue pelo catálogo de produtos
3. Use os filtros laterais para encontrar produtos específicos
4. Clique no ícone de coração para adicionar aos favoritos
5. Clique em "Adicionar" para incluir produtos no carrinho
6. Acesse o carrinho clicando no ícone no header
7. Finalize a compra preenchendo o formulário de checkout
